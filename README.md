# El Ghali Sany — Malaysia INCEIF MSc Islamic Finance Portfolio

Static GitHub Pages portfolio for an MSc Islamic Finance application.

## Contents

- `index.html` — complete single-page portfolio.
- `docs/` — evidence documents linked from the portfolio.
- `data/` — optional SUMO/GIS JSON folder for real traffic data.
- `.nojekyll` — prevents GitHub Pages from ignoring folders/files.

## Features

- English / Bahasa Melayu / Arabic switch.
- Arabic RTL support.
- Clean Malaysian green/blue institutional interface.
- SUMO / Leaflet traffic simulation layer.
- SC-SIIRS mini-readiness lab.
- Amanah Signal Easter Egg:
  - type `AMANAH` or `AMANAAH`;
  - or click the logo/name/title 5 times.

## Deploy on GitHub Pages

1. Create a repository.
2. Upload the full folder contents, not only `index.html`.
3. Keep `docs/` and `data/` at the repository root.
4. Go to **Settings → Pages**.
5. Select branch: `main`.
6. Select folder: `/root`.
7. Save and wait for deployment.
